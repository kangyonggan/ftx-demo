# FTX接口示例Java版

## 准备工作
- Jdk1.8
- Maven
- Git

## 配置
- FTX的apiKey
- FTX的apiSecretKey

## 示例功能
1. 通过ws订阅最新K线、盘口、最新成交。`WebsocketTest.java`
2. 通过rest api获取历史K线数据。`FuturesTest.java`
3. 获取账户信息。`AccountTest.java`
4. 下单、撤销订单、查询订单。`OrderTest.java`
